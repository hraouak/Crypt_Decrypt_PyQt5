Ce programme permet de crypter et décrypter une chaine de caractère ou un fichier texte.

Pour crypter un fichier texte veuiller créer un fichier texte au nom "decode.txt" où vous mettez votre texte à crypter
Pour décrypter un fichier texte veuiller créer un fichier texte au nom "code.txt" où vous mettez votre texte à décrypter
